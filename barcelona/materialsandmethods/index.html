Materials and Methods

To carry out this project, we used R Studio and Python to study air pollution in a scientific yet clear way. We started by cleaning the data because the original files were messy and contained information we did not need. Using tools like tidyverse, we filtered out the extra data and organized everything so the computer could perfectly understand the specific dates and hours of every record.

Next, we focused on fixing the table formats. We used a special script to turn technical sensor codes into easy-to-read columns, such as wind speed or the levels of each pollutant gas. Once the data was ready, we merged two different databases—one with weather info and one with pollution levels—into a single master table.
