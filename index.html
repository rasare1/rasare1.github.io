Darrera d'aquesta web...

Aquesta pàgina utilitza Bootstrap 5 per l'estructura i FontAwesome per la iconografia. Està programada totalment en HTML per mantenir una estructura física neta, mentre que el CSS s'encarrega de donar forma visual a cada paràgraf i llista.

La interactivitat de la web i el sistema de Mode Fosc estan gestionats amb JavaScript, permetent que l'usuari triï la seva experiència visual preferida.
