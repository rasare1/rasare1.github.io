Benvinguts als Projectes 🚀

Em dic Richmon ✌️, sóc alumne de robòtica i programació de 1r de Batxillerat de l'Institut Pompeu Fabra. Aquest és el punt de partida per editar tot i fer projectes amb el professor Francesc Pérez.

Estructura i Forma

Aquesta pàgina està escrita en html que dona l'estructura física i utilitza CSS que ens ajuda a donar forma a la web (paràgrafs, títols, llistes, imatges, links).

Interactivitat

A més utilitza JavaScript, que proporciona tota la interactivitat de la pàgina web, com el canvi de mode fosc i els menús dinàmics.

Bootstrap

Utilitza biblioteques externes com Bootstrap, que conté CSS predefinit i ens permet fer menús desplegables i dissenys adaptatius fàcilment.

FontAwesome

Incloem la biblioteca d'estil Font-Awesome per utilitzar icones i dibuixos vectorials que fan la interfície més intuïtiva i visual.
